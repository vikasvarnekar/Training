Parameters

argsObject
Type: System.CollectionsHashtable
A collection of name-value pairs where 'name' is interpreted as 'property name' and 'value' as the property value. Specified in the collection properties are set on the item prior to applying its AML.

Return Value

Type: Item
Returns an Item where the dom property is the XML returned from the server. The dom may contain an Item, a forest of Items (i.e. a disjoint union of AML trees), or an error.
Exceptions
