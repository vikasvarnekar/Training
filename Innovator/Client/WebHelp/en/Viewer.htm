Viewer Toolbar

The Sidebar in the item tear-off window contains the controls for the File Viewer Tabs. For more information, refer to Sidebar and its Properties.

The following toolbars are available through the viewer toolbar control:

  • PDF viewer toolbar

  • Image viewer toolbar

  • 3D viewer toolbar

The available commands are as follows:

Icon

Tooltip

Description

View

Displays the PDF file.

Markup

Enables you to annotate the file.

Highlight

Enables you to highlight portions of the file.

Label

Enables you to add a label to text or graphics in a file.

Select

Enables you to select items in a file.

Delete

Deletes selected item.

Download

Downloads a file.

Print

Prints a file.

Switch to Basic Toolbar

Switches to the Basic toolbar.

Switch to Standard Toolbar

Swtiches to the Standard toolbar

Zoom Up

Zooms closer

Zoom Down

Zooms farther away

Reset View

Resets to previous view

Orient to Face

Shows the CAD Model oriented towards you showing the selected part surface. 

Set Display Style

Selects the display style for the item.

Measure

Specifies the measuring method to use.

Exploded View

Enables you to explode the drawing to view the individual components.

Cross Section

Enables you to create and view cross sections of a drawing.

Model Browser

Enables you to view the model from different angles. There are two types of views: Standard and CAD.

Product Manufacturing information

Select this icon to view the associated product information.

Configurations

Displays a list of the possible viewing configurations for a CAD model.

Fit Width

Fits page width to the viewer frame in the horizontal direction.

Fit Height

Fits page height to the viewer frame in the vertical direction.

Page Up

Goes to the previous page (no effect if page 1 is current)

Page Down

Goes to the next page after the current one (no effect if last page is current)

Go To Page

This opens the Paging dialog:

   

   The current page number is shown.

   Type in the desired number and hit the Enter key to display the page.

   Click the top and bottom arrow icons to open the first and last page respectively.

Find

This opens the Find dialog:

   

   Type in the required search text and hit the Enter key to show the text.

   Click the right arrow to re-positions the file to show the next instance of that text string.

   Click the left arrow to search backward in the file.

   If the Highlight All checkbox is checked, all instances of the text are instantly highlighted.

   If the Match Case checkbox is checked, the text search is performed as case-sensitive.

Zoom Percent

This opens the Zoom dialog:

   

   The zoom value is set to 100% by default.   

   Enter the desired value to zoom the page to that percent.

Rotate 90 CW

Rotate file content 90 degrees clockwise

Rotate 90 CCW

Rotate file content 90 degrees counter-clockwise

Open Advanced

Open the Advanced toolbar.

Close Advanced

Close the Advanced toolbar.
