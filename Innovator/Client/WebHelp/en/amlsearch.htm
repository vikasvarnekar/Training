AML Search

This is a raw search mode and it uses the Aras Markup Language (AML).  Users with knowledge of AML can use this search mode.  It allows all users to make complex searches as well as use searches prepared by other advanced users, administrators, or developers and shared as Saved Searches.

AML Search can execute searches not possible with other search modes, including searches on:

In relationships and related Items

In source or parent Items for Relationship Items in the TOC

Using logic in multiple levels at the same time

Note: If you start building your search in Simple Search mode and then change the mode to AML, all of your criteria from the search mode will be converted to AML. This can save time instead of writing the search criteria from scratch.

For more information on AML, you can refer to the Programmer's Guide available in the Documentation section of the Aras web-site.  You can also attend training provided by Aras. For details on the training, refer to the Training section on the Aras web-site.
