About Data Types

When configuring properties for an ItemType, one of the columns is Data Type. Data Type is an attribute of every property of every Item in Innovator. The list of data types in the drop-down list of this field are listed below. We will explore each one in detail.

Data Type

Description

Boolean

Allows an answer of true or false, typically used with a check box on a form.

Color

A hex color value, automatically associated with a color widget for form selection.

Color List

To use this property, you must already have a list of colors to be used as a data source.
To create a list of colors see Creating a List. If you know how to create a list, and don't know the hex numbers to use for colors, you may refer to this http://www.lynda.com/hex.html# web site, recommended by Microsoft. If you specify the Color List as the data type, you must provide a data source.



Then, on the form, the list is represented with a drop-down box, like any other list:

Date

Value is date, selectable from the Date Dialog.  Dates are shown on the client a format determined by the user's Regional Settings for Windows, which can be set in Control Panel.

Decimal

Value has a precision (total number of digits) and scale (digits after the decimal point.)

Federated

A value to be provided from outside Innovator by a custom Method.

Filter List

A value filtered.

Foreign

See Foreign Data Type

Float

Value with scientific notation.

Formatted Text

Text with formatting which can be edited with the "Innovator Formatted Text Edito.r"

Image

An image, such as an icon.

Integer

a whole number.

Item

Another Innovator Item.

List

A predetermined set of values, usually displayed in a drop down box on a form.  For more information, please see Creating a List.

MD5

An encrypted value.

Multilingual String

A string with values in multiple languages, see Internationalization

MultiValue List

An Innovator List which can hold more than one value.

Sequence

A continuous series of numbers, incremented and formatted as specified in the Sequence properties.  See Sequences for more information.  

String

A string of letters and numbers of up to a length selected for the property.

Text

A long string of letters and numbers up to 1GB in length.

Variable

An Innovator Variable.
