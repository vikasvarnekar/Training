Add Forum

Click the Add Forum button  in the Forums section of the Bookmark panel to create a new Forum.

To create a new forum, do the following:

  1. Click the Add Forum button  in the Bookmarks Panel to open the Create new Forum dialog box.

  1. Enter a forum name in the text box.

  2. Click Enter to create the Forum. This forum is added to the bookmarks and is by default available to the World.

  3. To set different permissions or sharing settings on this Forum, right-click on Forum and select Share.... This opens the Share Forum dialog box:

 

 

    •  Public - Makes the forum  available to the World.

    • Private - Creates a private forum that is visible only to the creator.

    • Shared with Identities - Creates a list of identities to share a Forum with.

     For more information refer to the following section.

    Creating a Shared Identity Forum

    1. Enter an Identity name into the search box.

2.  Select the identities that you want to share your forum with.

3.  Click OK to save the changes or Cancel to exit from the Share with dialog box.

4.  Click OK to save the changes in the Share Forum dialog box or click Cancel to exit from this dialog box.
