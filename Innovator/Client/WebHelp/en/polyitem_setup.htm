PolyItem Setup

In Program Management, the Deliverable Item is defined as a PolyItem. It is up to the system administrator to determine which Item Types can be a Deliverable.

Setting up the Deliverable Item Type:

1.  In the Navigation pane, select Administration>ItemTypes. The following menu appears:

2. Click Search ItemTypes and open Deliverable for editing.

3. Click the Poly Sources tab.

4. Click the Select Items button .  The ItemType search dialog box appears.  Select any Item Types that will serve as deliverables.

5. Make certain that the selected ItemTypes selected have the correct permissions set up, i.e. that Project Management and All Employees have "get" access to each of these ItemTypes.

6. Click  to save and unclaim the ItemType.
