Adding Saved Searches to Forums

Saved Searches can be added to Forums, to drive dynamic content. Any Saved Searches for SSVC-enabled items can be added to Forums in Bookmarks.

Note: This functionality is limited to Saved Searches created by administrators only.

Use the following procedure:

1. Right-click on the context menu for a saved search and click Add to Forum. The Add Search to Forum dialog box appears.

2. Select one of the following options:

a. Create New Forum - selecting this option opens a dialog box as described in the Add Forum section. When you create a forum, the items you specify are added to it.

b. Selecting an existing forum that is owned by you adds the specified saved search to that forum.

Note: You cannot add a search to a forum that is owned by someone else.

3. Click Save and Done. The Saved Search associated with the forum appears on the Bookmarks tab.

Use the same procedure to add documents to a Forum.
