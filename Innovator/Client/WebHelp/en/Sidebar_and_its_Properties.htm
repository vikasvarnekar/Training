Sidebar and its Properties

When an ItemType is enabled for Secure Social, then the tear-off window for that item will automatically include a colored sidebar on the left side of the window that provides the secure social functionality.

All controls related specifically to Secure Social and Visual Collaboration are located in this sidebar.

  

Features of a sidebar include:

  • Discussion Button - A Toggle Control to show/hide the Discussion Panel.

  • Form Tab - Shown and enabled by default. This tab shows the ItemType icon of the item.

  • File Viewer Tabs - When the item has referenced files, the sidebar shows tabs for the appropriate files. Clicking on any of these tabs displays the selected file in the item window. Only one can file can be displayed at a time. When there is more than one file type available, clicking on the tab will open a drop-down list of all the available files of that type. The user can scroll down to select the desired file and open it in the viewer.

  • Item Keyed Name - The item's keyed name is displayed at the bottom of the sidebar.

  • All Tabs - Provides the user with a drop down list of all the associated files from which the user selects the desired file or form to open it in the tear-off window.

There are three kinds of File Viewer Tabs displayed on the sidebar:

Active

Inactive

Indication

Indicates one or more files to be viewed with the 3D viewer .

Indicates one or more files to be viewed with the PDF viewer .

Indicates one or more files to be viewed with the Image viewer .
