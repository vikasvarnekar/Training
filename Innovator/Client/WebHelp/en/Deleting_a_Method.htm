Deleting a Method

  1. In the Navigation pane, select Administration>Methods. The following menu appears:


  1. Click Search Methods. The Search grid appears. 
  2. Click  to see the method list.

 

 

  1. Right-click the method that you want to delete. The following context menu appears:

 

  1. Select Purge This Version to delete the most recent version of the method. Select Delete All Versions to delete all the versions of a method. A dialog box similar to the following appears:

  1. Click OK.

Note: When an error message appears after the confirmation pop-up window the item has not been deleted.  The name of the property corresponding to the ItemType appears in the error message as [itemtype_name property_name} and must be deleted from ItemTypes.
