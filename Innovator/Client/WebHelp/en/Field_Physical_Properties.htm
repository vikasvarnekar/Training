Field Physical Properties

Property

Description

Positioning

Absolute in window or Relative to top left of form. Absolute is recommended

X

Horizontal position in pixels

Y

Vertical position in pixels

Z-Index

Fields with higher Z-Index are drawn on top of Fields with lower Z-Index

Visible

Is the Field Visible

Disabled

Is the Field editable

Tab Index

Order in which tab key will move between Fields if Tab Stop is set

Tab Stop

Will the field be selectable using the tab key
