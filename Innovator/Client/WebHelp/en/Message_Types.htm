Message Types

There are different possible message types that can be posted within the Discussion panel comment box.

Message Classification

Description

Text Messages

A  comment/reply to a message with only text.

For more information on Manual messages, refer to Creating a Text Comment.

Snapshot Messages

A comment/reply to a message with a snapshot. It may or may not include text.

The thumbnail image in the message may or may not include markups.

For more information on Snapshot messages, refer to Creating a Snapshot Comment.

Audio Messages

A comment/reply to a message with audio.

Video Messages

A comment/reply to a message with video.

Automatic Messages

Automated messages are created by history updates to the item.

These message types and content are configurable by admin only.
