Manual Versions

Manual Versions is an optional capability that an Administrator can perform on an Item.  When an ItemType is defined by an Administrator and is marked as Versionable, the action of creating a new Version can be performed by the system automatically, through custom business logic, or manually, by a user.  When Manual Versioning is enabled, a Version menu option is available.  By selecting this menu option, a new version of the item is created.  

Manual Versioning can only be performed on items that are currently unlocked.
