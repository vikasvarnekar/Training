TOC Access Permission

Table of Contents (TOC) Access controls the access of ItemTypes in the main Tree View.  The following figure shows a new ItemType created and displayed in the TOC.  The Categories are the folders, or the tree nodes in the TOC.

The TOC access tab for each ItemType defines the category, or the tree node under which the ItemType will appear for each Identity.  Only users who are members of identities that have been specifically given TOC Access will see the node in the Tree View.  Users must be given TOC Access to an ItemType to be able to access the ItemType in any way.

TOC Access also controls the visibility of Categories in the Tree View.  When the user does not have access to any ItemTypes under a category (such as Administration),  the Category does not appear.  When the user is given TOC Access, but no category is specified, the ItemType appears at the top of the Tree.

Granting TOC Access:

1. Select TOC - -> Administration --> ItemTypes-->Search ItemTypes to select the ItemType that you want to expose in the TOC, and open it for editing. In this example, we will use Test.

2. Open the ItemType form and select the TOC Access tab:

3. Click Select Items. The Identity Search dialog appears. 

  to see a list of available identities. icon Search

4. Click the  icon to see a list of available identities

5. Select the Identity that you wish to allow access to this ItemType.  Remember that the only permission this grants is to access the ItemType in the TOC.  So, select the most inclusive Identity.  Click the green check mark to complete the selection.

6. Select the Category cell, and click the drop-down arrow to select the category where the ItemType will appear on the TOC for the selected Identity.

7. If you select Create Related and click the Create Item  icon , a new empty line is created.  Enter the name of a new Identity you would like to create and enter the Category where the ItemType will appear in the TOC for that Identity.

8. Click Save and Done to unlock and close the ItemType.
