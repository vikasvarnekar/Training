Can Add Permission

The Can Add permission allows the specific Identity to create or add instances of the ItemType.  Just like the TOC Access permission, this permission is added directly to the ItemType.  Without it, no one can create or add instances.

Setting the Can Add Permission

  1. In the Navigation pane, select Administration>ItemTypes. The following menu appears:

  2. Click Search ItemTypes to search for the ItemType you want to edit. In this example, we will use one called Sample. Open the ItemType for editing and select the Can Add tab.

  1. Clicking the Select Items button   enables, you to pick an existing identity to which you will add instances of the ItemType. If you select Create Item , you can create an identity on the spot to which you will give access rights for creating instances.
  2. Select Select Items. An Identity search dialog appears.
  3. Select the Identity to which you want to grant the Can Add permission, and click the green check mark icon .   A new row appears on the Can Add tab, which contains the name of the Identity that you just picked.  In the previous example, we picked the World identity, which means everyone.  To grant the permission for this identity, you need to check the box in the Can Add column.  If you uncheck the box, no permissions will be granted to the specified Identity.
  4. Click  to save and close the ItemType.
