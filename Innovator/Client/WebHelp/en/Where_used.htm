Where Used

You can use the 'Where Used' option in the context menu to view the list of Items that reference the current Item.  The Items are displayed in an expandable tree-structure format.

Let us consider a scenario where you would want to replace or remove a CAD document.  

Before making such decisions, you can view the Where Used tree structure to preview where the CAD document is currently being used and which parts or Items are affected by the change (replacement or removal).

Where Used functionality

 

  1. Log on to Aras Innovator.
  2. From the TOC, navigate to the Item. Let us consider a Part. Navigate to Design--> Parts-->Search Parts.
  3. From the main grid, select the Item and use the right mouse button (right-click) to open the context menu.
  4. Select Where Used. The Where Used pop-up window appears.

 

The Tree structure displays the related Parts and products.  Part C2951A-A1  is associated with four other Parts (C3801-80020-A.1, C9223-A1, and C9225-A.1). Part C9223-A.1 is also associated with Model 650A.1 and its associated Product. Part C9225-A.1 has a similar relationship with a Model and a Part. You can click twice on an Item to view the it.
