Creating a Sequence

Sequences are used for automatic numbering. They can be used to create comprehensive, unique identifiers for any items such as documents, parts, projects, etc.

To see the pre-defined sequences:

  1. In the Navigation pane, Administration>Sequences. The following menu appears:

  1. Click Search Sequences to see a list of existing sequences.

Creating a Sequence

  1. Click Create New Sequence. A blank Sequence window appears:

  1. Enter information in the following fields: 

Name - the name of the sequence item.

Prefix - the alphanumeric prefix of the generated number.

Initial Value - used to set the Current Value on import if none exists in the imported data.

Current Value - the initial value of the sequence.

Suffix - similar to Prefix, except added to the end of the number.

Pad With - specify the character to fill in the required number of spaces.

Pad To - the total number of characters for the number, excluding the prefix and the suffix.

Step - the increment by which the number differs from the previous and next.

The following examples show how these properties work to together to create a sequence configuration.
