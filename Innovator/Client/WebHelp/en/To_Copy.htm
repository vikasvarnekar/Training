To Copy

The copy process uses the Innovator Object Model (IOM) to obtain and display the following information for each relationship saved on the Clipboard:

  • Source id
  • Source type
  • Source keyed name
  • Related id
  • Related type
  • Related keyed name

The Copy process stores the original ID on the clipboard.  New ID's are assigned during the Paste process.

Copying from the Relationship Grid

  1. Highlight one or more items (rows) in the relationship grid.
  2. Right-click the highlighted rows and select {Tab Name}>Copy Row.  New clipboard entries are then generated for the copied item(s) and appear in the Clipboard Manager dialog.
