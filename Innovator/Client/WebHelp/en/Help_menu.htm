Help Menu

The Help menu provides options to view the online help, product version, and logged in session details.  The following table explains each option available in the Help menu.

Menu Option

Description

Just Ask Innovator

Displays the Just Ask Innovator online help.

About My Session

Displays information such as logged in username, name of database, and name of the server being accessed.

About

Displays the product version and current build information.
