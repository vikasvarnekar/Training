About Configurable Grids

Configurable Grids are part of the Innovator Framework, they are used in Quality Planning and Program Management Solutions and may be used in custom applications.  They are shown in the TOC in the Administration category with the name 'Grids'.

Basically a Configurable Grid is a client control which uses an AML query to retrieve data to populate the grid with data from the Innovator database.  A Grid definition contains the AML Query and Xpath expression to get data, a Method which runs when the grid is loaded, a specifications for the columns, an events to be executed when the grid is used.

Implementing Configurable Grids is an advanced developer function.
