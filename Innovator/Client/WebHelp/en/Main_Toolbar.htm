Main Toolbar

Aras Innovator’s user interface provides toolbar icons in the main window, tear off windows, and other dialogs.  

The following table describes the icons in the main toolbar.

Button Name

Button

Description

Edit

 Opens the Item for editing.

Save

Saves changes to an item. The item remains in Edit mode.

Done

Saves the item and unclaims it..

Discard

Discards changes to the item. The item returns to View Mode and is unclaimed.

Refresh

Refreshes the screen.

Promote

Allows you to promote the Item to the next level.
