View and Markup Modes

When a viewer is active in the item window, the user has access to the Viewer and Markup toolbars.

   

Only one mode is active at a time and it is indicated by a larger button.

View Mode

The user is in the View mode when the user clicks on the Viewer toolbar. In this mode the user can navigate through the file to look at the content.

Markup Mode

The user is in the Markup mode when the user clicks on the Markup toolbar. In this mode the user can draw markup notations on the picture.
