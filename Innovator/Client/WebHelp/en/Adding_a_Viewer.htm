Adding a Viewer

  1. Expand the Administration folder in the TOC.
  2. Expand File Handling and select Viewers. The Viewer window appears:

  1. Click Create New Viewer. The following window appears:

  1. Populate fields as indicated in the Viewer Properties topic.
  1. Click  and  to save and unlock the item.  The new action is now displayed in the default page in the main window.
