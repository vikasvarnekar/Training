Creating a Vault

During installation, Innovator creates a default file vault directory in the Innovator server in a directory typically named VaultServer. It also creates a virtual directory that provides access to the location on other computers from a web site. You must set the vault URL up to point to the Innovator server vault application file for the vault location. For more information about setting up a vault, refer tote Aras Innovator Installation Guide.

You can establish additional vaults in different locations or on different servers. Each vault location is referenced by a unique name and has its own virtual directory.

  1. Click Create New Vault. A blank Vault screen appears:

  1. Enter the vault name, URL pattern, and URL in the appropriate fields.

  1. Click the Search button  to search for the appropriate identity.
