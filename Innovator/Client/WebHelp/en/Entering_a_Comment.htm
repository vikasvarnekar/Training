Creating a Text Comment

The user can enter a text comment within the discussion panel when the item/form is active.

To enter a comment, do the following:

  1. Click the  Discussion button  in upper right corner  of the item window to show the discussion panel.

  2. Type in the required text in the Comment entry box within the Discussion Panel.

Note Use the Drag Handle to conveniently enter large text comments. The box can only be dragged lower while keeping the same width.

  1. Click the Comment button to post the comment.

Note: Pressing the Enter key does not post the comment, it creates a new line.

  1. Once the comment is entered, it is displayed within the content area of the Discussion Panel.

Components of a Message

Once the message is shown within the area of the discussion panel, there are various components in it that have specific properties.

The table provides properties for a message that is displayed in the discussion panel:

Components of Message

Properties

Source Icon

Indicates whether the source of the message is from a user comment, reply or automatic.

User name

Indicates the name of the user who created the comment, either directly or indirectly through History.

Creation time

Indicates the time when the message was created.

Referenced Item

Indicates the name of the item whose Board the message is related to. It acts as a hyperlink to open the tear-off window of that item.

Item info

The Item Info shows the basic item information from the item at the time the message was created. The displayed string is a combination of up to three properties from referenced item:

  • Current revision of the item

  • Current generation of the item

  • Current lifecycle state of the item

Item Icon

Icon and color corresponding to the referenced item.

Message text

The text of the message.

Thumbnail

The thumbnail of the snapshot (if included in the comment)

Message functions

The functions that can be performed on the message. For more details on message functions, refer to Message Functions.

Display of Messages

Use the scroll keys to scroll up and down to the bottom of the displayed messages.

When all the messages have been displayed, the message "There are no more messages" appears.
