The Mass Promote Grid

The Mass Promote tab displays the items that you selected to promote. It consists of the following components:

  • The Banner displays the Mass Operation Name (Mass Promote), the number of selected items and their ItemType.

  • The Settings Panel dynamically displays information about the current state of the Promote operation. For more information, refer to Mass Promote Setting Modules.

  • The Toolbar displays the standard icons found in the main and relationship search grids.

  • The Grid resembles the search and relationships grid used in Innovator, but it is specifically built for Mass Promote operations.

The grid contains a fixed set of 5 columns:

The following table describes the columns:

Column Description
Lock A green flag  appears to indicate that the item is locked.
Item The keyed name property of the item.

The column label must be "Item" and not "Keyed Name."
Current State The State property of the item. State refers to the target state for the items being promoted.
Valid Click the checkbox for the item(s) that you want to promote.
Status/Error Displays either status or error messages for the items.

You can only do a simple search in the grid.

Once you have selected items from the main grid and populated the Mass Promote grid, you can add and remove items before you begin the promotion process.
