Secure Social Visual Collaboration for Item Windows

The purpose of Secure Social and Visual Collaboration (SSVC) is to deliver easy-to-use social collaboration and view markup capabilities within Aras Innovator.

Secure Social provides a powerful, configurable mechanism to aggregate social content, including files, to related items.

For example, Parts can include discussion content and files from related documents and CAD documents. This allows social content to be viewed from multiple contexts, increasing its value.

Functionality for Secure Social and Visual Collaboration can be described in two distinct but related areas:

  • “Secure Social ” that provides threaded discussions across all Aras Innovator items.
    Refer to the topics under     Secure Social Functionality for more information.

  • “ Visual Collaboration” that provides web-based viewing and markup.
    Refer to the topics under Visual Collaboration Functionality for more information.
