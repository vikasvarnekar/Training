Structure Browser

Each Aras Innovator ItemType can contain one or more relationship instances. The Structure Browser is used to rapidly view relationship instances in an expandable format. This type of viewing is most useful with part structures such as a Bill of Materials or the viewing of complex relationships between Administration ItemTypes. The examples to follow use a simple Bill of Materials to show the simplicity of the Structure Browser. In general, to view any ItemType in a Structure Browser window, simply Right-Mouse-Click (RMB) an item in the search grid and select “Structure Browser” in the selection list.

The chosen item displays its ItemType – Name – Revision.Generation as a single line in a Structure Browser window.

Tool bar icons

Toolbar icon

Description
Expands all levels of a structure.
Collapses all levels of a structure.
Does a comparison with another item of the same type. For example, a Part item can be compred to another Part item or an Administration ItemType can be compared with another similar Administration ItemType. A comparison of this type compares all relationships existing on each item.
Does a comparison with another version of the same item.  For example, a Part item can be compared to another version of the same Part having similar configuration IDs. A comparison of this type compares two versions of the same ItemType that are defined as Versionable (see About Versions for more information on creating a versionable ItemType). When this icon is selected a search dialog appears with all prior generations.
Expands entire structure and displays item differences in color.

Performing a Side-by-Side compare with another like ItemType

  1. From the Structure Browser select the Side-by-Side with other Part icon  to search for a Part to compare relationships. In this example we only compare the Part BOM relationships, other relationships have been removed for clarity.

  2. A new popup window displays the parts from which a comparison part can be chosen. You can search in this window for a desired part using basic search techniques.

  1. Select the part item to compare and click the green check iconto complete the selection. The chosen part splits the Structure Browser into two separate relationship hierarchies.

Each side of Structure Browser is fully expandable.

Highlighting differences of this Side-by-Side compare

Click on the Highlight iconto see color coded differences.

The following is a description of the Highlight color legend:

  • BLACK – Indicates no difference between items in structure or version/generation. Root item types always appear black since the difference is known.

  • RED – Indicates a difference in generation between two items of matching base number.

  • BLUE – Indicates no item was found with a matching structure. A Part BOM relationship to related part items California and BR-0201 are not found under BRAKE-02.

Performing a Side-by-Side compare with another version of an ItemType

From Structure Browser select Side-by-Side with other Version icon  to search for a Part, to compare relationships. In this example we only compare the Part BOM relationships, other relationships have been removed for clarity.

  1. A new popup window displays all versions of the part item to choose from.

  1. Choose by double-clicking a version. In this example version 1 is selected. See About Versions for more information on Innovator versions.

The item version comparison is showing part item California was added to BRAKE-01 Revision A Version 2

Highlighting differences of this Side-by-Side compare

The item version highlight comparison shows that there is no matching structure for Part-Caliper-1 - A.1. A brief description of the color legend as shown in this example follows.

Click on the Highlight iconto see color coded differences.

  1. BLACK – Indicates no difference between items in structure or version/generation. Part BOM relationship to related part item BR-0301 is found to match in structure and version/generation.Root item types always appear black since the difference is known.

  2. RED – Indicates a difference in generation between two items with matching structure. Part BOM relationship to related part item BR-0101 is found to match in structure, but generation differs between the two.

  3. BLUE – Indicates no item found with matching structure. Part BOM relationship to related part items California and BR-0201 are not found under BRAKE-02.
