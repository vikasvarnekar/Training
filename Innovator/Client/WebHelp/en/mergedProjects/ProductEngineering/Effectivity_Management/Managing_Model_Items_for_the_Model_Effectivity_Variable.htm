Managing Model Items for the Model Effectivity Variable

The Model Effectivity Variable uses items of the Model (effs_model) ItemType. There are no out of the box Model items. If you use the Model as an Effectivity variable, you need to create Model items.

Select Administration>Effectivity Services>Models in the Aras Innovator TOC to create/access Model items.

Creating a Model Item

  1. Select Create New Model from the Models menu. A blank Model form appears:

  1. Enter a name and label for the model and click . The Name field is required and should be unique.

If a Model has a label, then the label is visible to a user working with effectivity. If the Model does not have a label associated with it, the Name appears.

Editing a Model Item

Before changing the Name or Label of a Model item, make sure that the Model item is not used in any Effectivity Expressions.

  1. Select Administration>Effectivity Services>Models from the TOC. The Model menu appears.

  2. Click Search Models to see a list of existing Models.

  3. Double-click on a Model in the Search grid and click  to modify the Model.

  4. Click Done to save your changes and unclaim the Model.

Deleting a Model Item

Before deleting a Model item, make sure it is not used in any Effectivity Expressions.

  1. Right-click the appropriate Model to open the drop-down menu.

  1. Click Delete All Versions. A dialog box similar to the following appears:

  1. Click OK.
