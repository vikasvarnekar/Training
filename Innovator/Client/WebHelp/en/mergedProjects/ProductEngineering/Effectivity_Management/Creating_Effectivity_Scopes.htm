Creating Effectivity Scopes

Use the following procedure:

  1. Select Administration>Effectivity Services>Effectivity Scope in the TOC. The following menu appears:

  1. Click Create New Effectivity Scope. The following form appears:

  1. Enter the name of the effectivity scope in the Name field.

  2. Click the ellipses in the Builder Method field and select effs_scopeObjectBuilderMethod unless you are implementing a different method that uses custom business logic.

The Name and Builder method fields are required.

  1. Specify Effectivity Scope ItemTypes and variables on the corresponding accordion tabs.

  2. Click  to save and unclaim the Effectivity Scope.

Finding Where an Effectivity Scope is Used

Click the Navigate button  in the toolbar and select Where Used from the dropdown menu to access the Where Used browser.
