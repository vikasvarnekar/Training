Creating a DCO

DCOs can be used to release, change, revise, or obsolete Affected Documents. The DCO creation process can be divided into general and specific procedures. The following procedure shows the general steps to use to create a DCO. For information about releasing, changing, revising, and obsoleting a DCO, refer to the Aras Innovator Product Engineering 12.0R1 User's Guide.

1. Click the Navigation button . The Table of Contents appears.

2. Select Change Management>DCOs. The following menu appears:

3. Click Create New DCO. A blank DCO form appears.

4. Enter the DCO title as precisely as possible in the Title text box.

5. Describe the reason for the change as precisely as possible in the Change Reason text box.

6. If it is necessary to designate an Express DCO user other than yourself, use the following procedure:

a. Click the ellipses in the Change Coordinator field. The Search dialog - Identity dialog box appears.

b. Search for and select a Team. This Team replaces the default identity.

7. If it is necessary to select a Team other than the default:

a. Click the ellipses in the Team box. The Search dialog - Team dialog box appears.

b. Search for and select a Team. The specified Team replaces the default identity.

8. Describe the change as precisely as possible in the Change Description text box.

9. Specify the change priority using the Priority radio button: 1-High, 2-Normal, 3-Low.

10. Keep the Release Date box blank. This box is automatically populated with the date when the DCO is approved and closed.

11. Click the Attachments tab to attach all of the files related to the DCO.

12. Click the Create Item icon  in the Attachments toolbar. The standard operating system dialog for file browsing and selection appears.

13. Search for and select a file. A new row appears in the Files grid with the file attached.

Aras Innovator automatically provides a file type in the File Type cell depending on the File's extension when you save the DCO.

14. Enter any additional information about the file in the Comments cell.

15. Click Done. The DCO is created and waits to be either submitted or cancelled.
