Obsoleting a Document within a DCO

There is a difference between obsoleting and deleting a document. The DCO Delete action obsoletes a Document. It does not delete either the Document or its attached files. It informs users that the Document is Obsolete and should no longer be used.

Use the following procedure to assign an existing Affected Document to be deleted when creating a DCO:

1. In the Affected Items tab, select Delete from the dropdown list in the Action cell.

2, Click the ellipses button in the Old Number cell. The Search dialog - Change Controlled Item appears.

3. Search for and select the Document to be deleted. The selected affected Document populates the Old Number cell.

4. Click  to save and unclaim the Document. The DCO has an affected Document assigned for deletion.
