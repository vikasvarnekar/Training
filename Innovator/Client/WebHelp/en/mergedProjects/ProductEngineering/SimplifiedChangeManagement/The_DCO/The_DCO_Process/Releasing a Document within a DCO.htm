Releasing a Document within a DCO

Use the following procedure to assign an affected Document in the Preliminary state for release when creating a DCO.

1. Open the DCO View.

2. Click .

3. Go to the Affected Items tab and click the Create Item icon . A new row appears in the Affected Items grid.

4. Select Add in the Action cell.

5. Click the ellipses in the New Number field cell. The Search dialog - Change Controlled Item appears.

6. Search for and select the new affected Document. The affected Document populates the New Number cell.

7. Click Done. The DCO has an Affected Document in the Preliminary state assigned for release.
