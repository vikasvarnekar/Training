Configuring Sequences

Sequences are used for automatic numbering of PRs, ECRs, and ECNs. Sequential numbering is an option for Parts and Documents. This section shows how to configure the appearance of a sequence to comply with your company's requirements.

Viewing pre-defined sequences

  1. Click the Navigation button , and select Administration>Sequences. The following menu appears:

  2. Search as usual, and you should see the following available sequences: Document, Part, ECN, ECR, and PR.

Let's use the ECN sequence for an example.

Configuring a sequence

Open an ECN sequence for Editing. You should see the Sequence form like this:

 

Here is a list of the properties and their contribution to the sequence configuration:

  • Name - the name of the sequence item.

  • Prefix - the alpha-numeric prefix of the generated number. For example, by default, all
    ECNs have an ECN Number: ECN-10000x.

  • Suffix - similar to Prefix, except added to the end of the number. For example, if you were to specify a suffix of - GTX, the ECN number would look like: ECN-100001-GTX.

  • Pad With - specify the character to fill in the required number of spaces.

  • Pad To - the total number of characters for the number, excluding the prefix and the suffix.

  • Step - the increment by which the number differs from the previous and next.
