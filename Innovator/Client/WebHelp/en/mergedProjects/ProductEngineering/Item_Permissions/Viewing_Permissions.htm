Viewing Permissions

An Item Permission defines which access rights each user role has for performing actions on Items at a particular Life Cycle state. Your ability to add, edit, delete, view, and search Items depends on the Item Permissions set by an administrator for you.

Use the following procedure to view the current permission for an Item:

  1. Go to the Item view.
  2. On the Item toolbar, click  and then select Permissions. The Permissions submenu appears. The current Item permission displays a blue checkmark.

  3. Click View. The view of the current Item Permission appears showing the Access accordion with identities and their rights.
