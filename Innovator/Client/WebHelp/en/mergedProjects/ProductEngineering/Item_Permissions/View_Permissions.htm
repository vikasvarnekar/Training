Viewing Permissions

Item permissions define which access rights each user role has for performing actions on Items at a particular Item Life Cycle state. Your ability to add, edit, delete, view, and search Items depends on the Item Permissions set by your administrator.

Use the following procedure to view the current Permission for an Item:

  1. Go to the Item view.
  2. Click the More icon  to view the following dropdown menu:

  1. Select Permissions>View. The current Item Permission view appears showing the Access accordion with Identities and their rights.
