Creating Private Permissions

A Private Permissions is a Permission set specifically for a given Item that overrides the current permission. For example, Private Permissions can be set for confidential Items to prevent them from being viewed by other identities. You can change the access rights defined by the current Item Permission by creating a Private Permission for the item. This Private Permission is saved as a separate Permission item applied to a specific item. Once it is set, the Private Permission controls access to the item by imposing a new set of access rights.

You can roll back to the default Item Permission, which deletes the Private permission. The Can Change Access grant in an Item Permission defines who can create Private Permissions. The Item and Assigned Creators have this grant by default. Administrators define who can change Item access rights. The following procedure assumes that the user has change access rights.

 

  1. Go to the Item view.

  2. Click  and . 

  3. Select Permissions>Create Private from the dropdown menu. The view of the new Private permission appears pre-populated with the grants definition from the default Permission.


  4. If necessary, do the following:

    • Add existing identities to the Private Permission.
    • Create identities and add them.
    • Replace Identities.
    • Remove Identities.
    • Select the appropriate check boxes in the Access grid to define who has which grants.
    • Click  to create the Private permission.
  5. Return to the Item View and click . 

  6. Select Permissions. Create Private is disabled and Current Item Permission no longer has a blue checkmark next to it.

Adding Identities to Private Permissions

Use the following procedure:

  1. Click the Select Items icon on the Access accordion toolbar in the Private Permission view. The Identity Search dialog appears.

  2. Search for and select the necessary identity, which appears at the bottom of the access grid.

Creating Identities for Private Permissions

Use the following procedure:

 

  1. Click the Create Items icon  on the Access accordion toolbar on the Private Permission view. A new row appears at the bottom of the grid.

  1. Enter the identity name in the Name cell of the new row.
  2. Save the Permission. The new identity is created.

Replacing Identities in Private Permissions

Use the following procedure:

 

  1. Click , right-click the identity to be replaced and then click Replace. The Identity Search Dialog appears.
  2. Search for the appropriate identity to replace the existing one. The grants are retained for the selected identity.
