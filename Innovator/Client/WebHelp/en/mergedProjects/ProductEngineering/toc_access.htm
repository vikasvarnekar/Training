TOC Access

Depending on the membership of the user in an Identity (e.g., CSI or Component Engineering), each user will see different folders on the main navigation tree. Currently there are only two identities that control these settings - All Employees and World. For each of the ItemTypes in this application, an administrator can alter the TOC Access permission which controls the visibility of the business objects.

As shown here, only the Employees get to see the Sourcing folder, as well as the ECN folder of Change Management.
