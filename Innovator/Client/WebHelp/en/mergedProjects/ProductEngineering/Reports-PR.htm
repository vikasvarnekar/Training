Reports-PR

No matter what activity the PR is entering, what state the PR is in, or who created it,  the CSI has access to all PRs in the system. The CSI can view the following reports about the PRs:

  • PR log report - shows a list of all the PRs, their current status, who created them, and who is assigned to each.
  • PR report - a printable view of the PR form.

These reports are available by selecting Administration>Reports from the Navigation pane.
