Selecting when Project is Updated

  1. In Release 8.1 there are tailorable choices to determine when Projects are updated.
  2. Selection can be made for each individual Project from a List on the Project From
  3. Alternatively all Projects may use a default selection and hiding the List on the Project Form
  4. The selection choices use the same list as for Selecting when a Project is Scheduled
    1. Interactively only
    2. Interactively and on Unlock
    3. On Activity Completions
    4. On Activity Updates

To change when an individual Project is Updated

  1. On the Project form users may make a selection from the dropdown labelled Project Update Mode

To use the same selection for all Projects

  1. This change can be made by a member of the Administrator Identity
  2. Edit the Project ItemType
  3. Set the required default value for the update_mode Property
  4. Edit the Project Form
  5. Change the update_mode Field to Visible = unchecked on the 'Field Physical' tab
