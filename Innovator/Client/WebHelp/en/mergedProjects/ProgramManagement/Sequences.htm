Project Number Sequence

When a new project is created, its project number is automatically generated by a sequence. The Project Number is used internally by the Program Management Solution code, the Sequence should not be edited otherwise errors will occur. For customers who would like to number Projects with a Sequence of their choosing, a new Property can be added to the Project ItemType and the Form edited to hide the project_number Property and display the added Property of data-type Sequence.
