Create a Project, Documents

The documents tab is designed to hold documents that relate to the whole project, usually some inputs such as design specs or maybe even meeting notes and presentations. However, this could be any Deliverable, not just a document. What exactly is stored here, is typically determined by the company's best practices.

To attach a deliverable (document) to a project:

  1. Open the project for editing.
  2. Select the Documents tab. Select Pick Related (if attaching an existing Deliverable), or Create Related if creating a new document or Deliverable.
  3. Click on the New Item icon . If picking an existent item, a search dialog will open, listing all of the deliverables.

  3. Save, Unlock, and Close the Project.
