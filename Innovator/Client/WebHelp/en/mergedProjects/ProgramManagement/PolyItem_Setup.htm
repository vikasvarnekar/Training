PolyItem Setup

In Program Management, the Deliverable Item is defined as a PolyItem. It is up to the system administrator to determine which Item Types can be a Deliverable.

 

To set up the Deliverable Item Type:

 

  1. In the TOC, select  Administration>ItemTypes. Search for and open Deliverable for editing.
  2. Select the Poly Sources tab.

  1. In the Actions, select Pick Related.
  2. Click the Create Item icon  . A search dialog will be displayed. Select any Item Types that will serve as deliverable.
  3. Make certain that the individual Item Types selected have correct permissions set up, i.e. that Project Management and All Employees have "get" access to each of these Item Types.
  4. Save, Unlock and Exit from the Deliverable Item Type.
