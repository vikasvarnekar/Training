Create a Project, Team

The second tab in the Project form is the Team tab. This tab is useful only if the Project is being created from a template. When creating a project from a template, there are no assignments for the activities. Instead, each activity has a role designation. Once the template is used to create a project, the resultant project will have all the roles listed under the team tab.

 

 For more information about this tab, go to Create a Project from Template.
