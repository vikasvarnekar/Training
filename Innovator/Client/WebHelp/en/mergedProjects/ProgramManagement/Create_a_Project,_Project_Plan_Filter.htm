Create a Project, Project Plan Filter

The Project Plan Grid has a blue Filter bar at top, this also present in the Project Template

Clicking the column headings in the Project Plan grid does not sort the columns; sorting a mixture of WBS Elements and Activities does not produce meaningful results.

 

To filter the Project Plan

  1. Enter search terms in the blue Filter bar, wildcard characters * and % may be used
  2. Click the Filter icon in the toolbar
  3. Activities which match the filter criteria are displayed in the Project Plan grid with their parent WBS Elements as shown below
  4. If no Activities match the filter criteria the Project Plan grid will be blank

 

  1. To restore the unfiltered Project Plan, click the Clear filter Icon
