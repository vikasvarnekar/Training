New Kitchen Template

Let’s create a template for the New Kitchen example we used in the Getting Organized section of the PM Workshop.

We’ll use the same plan for our Template. To make use of the out-of-the-box ‘Project Team’ List we’ll use the Manager role for ‘Me’ the kitchen owner, the Assemble role for Jerry the contractor and the Process role for Bob the decorator. Here is the Project Template.

The project plan includes the following:

  • WBS Elements and Activities have been created.

  • Precedence is shown in the Predecessor column.

  • Duration in days for each Activity is shown in the Planned Duration Column.

  • The Lead Role is also shown.

No assignments were created for Activities because the first Walkthrough will not use assignments.

The template has been saved, unlocked and promoted to Approved so that Projects can be created from it.
