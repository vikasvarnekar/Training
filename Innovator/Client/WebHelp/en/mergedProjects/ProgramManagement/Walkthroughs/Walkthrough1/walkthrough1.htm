Walkthrough 1, Activity Leader only

In this walkthrough we’ll use the simplest case, Activities will have a Leader but there will be no assignments, no work estimate and no time reporting. The Project Manager delegates responsibility to a single Leader for each activity and the Activity Leader will report its completion.

 

  1. First we will create a project from the New Kitchen Template we just created, using the steps shown in Create Project from Template.

  2. Save the Project to create a schedule as shown here.

In the screenshots for this walkthrough, holidays have been added to the Business Calendar and some Activities scheduled within their float.)

  1. Note that details from the Template have been copied to the Project, including the Lead Role, but there are no Leaders yet. If we look at the Team tab we see that the Project Team has roles for Manager, Assemble and Process and that we can Pick/Replace alias Identities to fulfill each Role. By doing so the Project Team will look as follows:

 

  1. When we save the Project, now the Project looks as follows, Leaders are now showing in the Project grid.

 

  1. With the schedule set according to the Project Manager’s needs, and the team selected, we are now ready to promote the Project to Active and to update it to activate Activities and show status. The result is shown below:

  1. Note that there are two active Activities which are on the critical path, Fridge and Sink. So our next step when we have placed the orders is to report our progress in the InBasket, which looks as follows:

  1. Note that the Sink and Cabinets Activities show in my InBasket. Let me go ahead and report progress for ordering the Sink first, the Activitiy Completion Worksheet (ACW) looks as follows:

  1. Note that there is no Assignment tab because there are no Assignments, which was our intent in this walkthrough. There is a Time Record tab which we will ignore because we do not intend to uset time reporting ( the Time Record tab can be hidden if desired, see Tailoring and Customizing).
  2. Two boxes on the form are highlighted, indicating that I as the leader can report Actual Start Data and Percent Complete. Since this activity has duration 40, I’ll report 1 day out of 40, 2% complete with a start date of today, and click the Save button in the Toolbar. The activity remains in my InBasket. The activity will not be complete until the sink is received, which is when I would use the Complete button.
  3. I’ll do the same thing for the Cabinets activity, reporting 1 day out of 30, 3% complete with today as the Actual Start Date.
  4. After updating the Project it now looks as follows:

  1. Note that the percent complete has rolled up to the Buying WBS, Phase 1 and to the root WBS as well.
  2. Reporting further completions will work the same way for all the other Activities.
