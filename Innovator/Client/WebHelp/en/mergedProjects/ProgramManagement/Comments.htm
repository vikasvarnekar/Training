Comments

Activity Comments Tab - this tab holds any commentary and files that may be necessary to complete an activity. For example, if a template is necessary to write a functional specification, it could be included in the comments, along with any other file, such as a CAD drawing, or an Excel spreadsheet. Here is a view of the information in this tab:

 

To add a new comment, click on the New Item icon , and a new line will be added to the table.

Fill out the following properties:

  1. Created By - the identity responsible for this comment, defaults to who ever is creating this item
  2. Created On - the date the comment was created, defaults to when the line item is created
  3. Comments - the actual comment or description
  4. File Attached - the link to attach a necessary file, if any.
