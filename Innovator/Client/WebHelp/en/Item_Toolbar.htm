Item Toolbar

The Item window provides a toolbar with buttons to perform various tasks.  

 

The following table describes the buttons in the Item toolbar.

Button Name

Button

Description

Create New Item

Creates a new Item. The button name changes according to the type of item you want to create.

Save

Saves the open Item to the server.

More

Click the More button to see the Context menu. From there you can select Purge this version to delete the current version of the item or Delete All versions.

Refresh

Refreshes  and reloads the Item information on the Form.

Export to Excel

Click the Share button and select Export to Excel from the context menu.

Export to Word

Click Share to export all the Item properties and relationship properties of the open Item to a Microsoft Word document.

Copy ID

Select Copy ID from the context menu to copy the ID to  the clipboard.

Copy Link

Select Copy Link from the context menu to copy the specified link to the clipboard.

Copy Link To

Select Copy Link To from thecontext menu to copy the specified link to either the latest version or the latest release.

Claim

Claims the Item (Edit mode) and prevents other users from submitting changes.  For more information, refer to Claiming or Unclaiming an Item.

Unlock

Unclaims the open Item and allows any authorized user to lock the Item for change.  For more information, refer to Claiming or Unclaiming an Item.

Undo Changes

Reverts the Item back to values since last saved.

Promote

Promotes the Item to the next available state, defined in the assigned Lifecycle map.  For more information on promoting an Item, refer to Promoting an Item in Lifecycle.

Navigate

Click Navigate and select Versions from the context menu. The version information for the selected item appears.

Hide Tabs in Tab View

Toggles display of the Relationship tabs.

Help

Displays the Just Ask Innovator online help.

Done

Saves the changes made to the Item, unlocks the Item, and then closes the Item window.
