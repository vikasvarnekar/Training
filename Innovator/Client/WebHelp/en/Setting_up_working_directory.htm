Setting Up Working Directory

A working directory is the location on the local client machine for files that are retrieved from the Vault (during a check out operation).

When you log on to Aras Innovator, you are prompted to set up a working directory, if the working directory specified for you is not valid.  You can select the required location and configure your working directory.

If the administrator has correctly configured the working directory for the end users, the prompt to set up a working directory may not be displayed.  However, you can always change the location of the working directory.

To set up or change the working directory

  1. Log on to Aras Innovator.
  2. Navigate to Tools --> Preferences --> Change Working Directory.

The Browse For Folder dialog box is displayed.

Figure 1   

  1. Select the required location for the working directory on the client machine.

You can click Make New Folder to create new folders on the client machine.

  1. Click OK.

The specified location is the working directory on the current client machine.
