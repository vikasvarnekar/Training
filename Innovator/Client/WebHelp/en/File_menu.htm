File Menu

The File menu provides various options to perform basic tasks such as create, save, open, export to Microsoft Excel or Word, and so on.  The following table explains each option available in the File menu.

Some options in the File menu may appear disabled.  The menu options are enabled on appropriate Item selection and if the logged-in user has adequate access rights.  The access rights are configured by the administrator.

Menu Option

Description

New

Creates a new Item.

Save

Saves the selected Item to the database.

Save As

Allows you to create a duplicate copy of the selected Item.

Open

Opens the selected File Item in view-only mode.

In most cases, this option is enabled only for administrator users.

Download

Downloads a copy of the selected File Item to the working directory.

In most cases, this option is enabled only for administrator users.

Check In

Checks in the file from your working directory and unlocks it.

Check Out

Checks out the current file to your working directory and locks it.

Check Out to Directory

Checks out the current file to a directory you specify.

Undo Check Out

Reverts the file to the version before checkout.

Printable View

Prints active window to client print driver.

Export to Excel

Exports the Item properties listed in the main grid to a Microsoft Excel sheet.

Export to Word

Exports the Item properties listed in the main grid to a Microsoft Word document.

Logout

Logs out of Aras Innovator session or closes open window (applies only to tear-off windows).
