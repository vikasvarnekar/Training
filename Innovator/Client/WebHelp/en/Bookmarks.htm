Bookmarks

Bookmarks represent social content you are following. The discussion content of the selected bookmark is shown in the Discussion Panel.

Default Bookmarks

Default Bookmarks are automatically set and available for all users:

  • All Bookmarks:  This is the default bookmark which is active when you open My Discussions. It shows aggregated content from all bookmarks at once.

  • Messages from me:  All messages that you have created (This does not include History messages indirectly created by the user)

These bookmarks are always shown in this order at the top and cannot be collapsed.

Bookmark Groups

Bookmarks that the user can create/add are categorized into 4 bookmark groups:

  • People:  Bookmarks that represent all messages created by a specified person (does NOT include History messages indirectly created by the user)

  • Forums:  Bookmarks representing Forums

  • Items:    Bookmarks representing individual items

Each of these groups has a header within Bookmarks that allows the group to be displayed or collapsed.

Bookmark Search

The Search Bookmarks field appears at the top of the Bookmarks panel. Enter your search criteria in this field and click on the icon. Search is done by searching on the entered search string within the names of the bookmarks.

When search results are in effect, only the bookmarks matching the search criteria will be displayed in the discussion panel.
