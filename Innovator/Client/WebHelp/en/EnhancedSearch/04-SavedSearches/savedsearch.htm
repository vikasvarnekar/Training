Saved Search

The Saved Search ItemType enables users to retrieve searches that have been created previously; they can be used in several ways.

Saved Search for Users

  • After you perform a Search you can save it, giving it a name, so that you can retrieve and re-use it, with or without modification, in the future.

  • Whenever you perform a Search the Search Mode and search terms are saved automatically. When you return to an item the last search is restored automatically.

  • Every user can execute Saved Searches they have made, in addition to searches shared by other users.

 

Saving a search

  1. Make a search in any Mode.
  2. Click the Add to Favorite button  to save the search. The Favorite Search dialog appears.
  3. Enter a name for the search in the Label field.
  4. Click the ellipses in the Share with: field to share the search with other Aras Innovator users in your organization.
  5. Click the Pin to Quick Access checkbox to display the shortcut icon   for the search in the TOC.
  6. Click Save. You can access the search by selecting Administration>Saved Search in the Navigation panel.

 

Retrieving/Applying a Saved Search

  1. Log onto Aras Innovator.
  2. Select Administration>Saved Searches in the TOC. The Saved Searches grid appears. The Saved Search menu appears:

  3. Click Search Saved Searches to see a list of saved searches.
  4. Right click the search that you want to execute and click Run.

Deleting a Saved Search

  1. Log into Aras Innovator.
  2. Select Administration>Saved Searches in the Navigation pane. The Saved Searches grid appears.
