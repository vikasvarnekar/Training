Message Search and Filtering Options

The Discussion toolbar contains options to easily search and filter the messages using various criteria.

There are two ways of performing a message search in the Discussion toolbar.

Basic Search

Basic searching, which searches on message text only, is provided directly in the toolbar.

To perform a basic search of messages, do the following:

  1. Enter the search text in the Search entry field.

  2. Click the Search icon to execute the search. All the messages containing the search text are populated within the discussion panel.

  3. Scroll down to the bottom to find all the messages displayed in the discussion panel.

Advanced Search

Advanced Search searches on the values you enter in the filtering options provided.

To perform an advanced search of messages, do the following:

  1. Click on icon located in the search entry box in the discussion toolbar. This opens a floating window below the search box.

  1. Enter values in any of these filtering criteria. The options are described in the following table:

Filtering Criteria

Filtering Options

Type

Message search is based on the type of message:

  • All comments

  • Text comments only

  • Snapshot comments only

  • Audio comments only

  • Video comments only

  • Automated messages only

Source

Message search is based on the source of message:

  • All

  • This item only

From

Message search based on the creator of the message..

Timeframe

Message search based on the Timeframe from when the message was created. There are two options:

  •  "Within the last" allows you to specify a number and a time value (Minutes, Hours, Days, or Months)

                                              

  • The  "Date range" option enables you to search for messages that were created during a specific time period. 

                                                 

When you click Search, a message similar to the following appears:

Revision

Message search is based on the Revision value of the item.

State

Message search is based on the name of the state of the item.

  1. Click on Search to execute the search to display the filtered messages.

  2. Click on Cancel to close the Search window.
