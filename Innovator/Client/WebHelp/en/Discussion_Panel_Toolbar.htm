Discussion Panel Toolbar

The toolbar on the Discussion Panel has various controls for entering comments and controlling the display of the messages within the panel.  The function of each part of the toolbar is described below:

The following table explains the properties of the parts in the discussion panel toolbar

Parts of Discussion Toolbar

Properties

Add comment/Comment entry box

Create comments within the Discussion Panel by typing text into the Comment entry box and clicking the Comment button.

The comment is displayed within the content area of the Discussion Panel.

Drag Handle

Drag Handle allows you to conveniently enter large text comments.

The lower-right corner of the comment entry box allows the user to drag the box to a large size.

Attach Dropdown

Click the Attach link to attach a sketch or picture to your comment. This shows up only when the viewer is in the markup mode. It is not shown when a form/viewer on view mode is active.

For more details on the markup mode, refer to View and Markup Modes.

Message Sorting Options

There are three kinds of message sorting options:

  • Conversation

  • Thread Date

  • Message Date

   For more details on the message sorting options, refer to Message Sorting Options.

Message Display Options

Changing this selector will change the style of all comments in the Discussion.  There are two modes of display:

  • Standard - Standard mode is the normal mode which is the default mode that is selected..

  • Review  - Review mode is for reviewing markups. In this mode, all the markup messages are displayed as a "snapshot style".

Basic search Box

Type in any word in the search box to start searching for specific messages.

For more details on the basic search options, refer to Message Search and Filtering Options.

Advanced Search Options

This provides advanced message search options.

For more details on the advanced search options, refer to Message Search and Filtering Options.

Share Comment With

Enables you to share a comment with a single person or group of people.
