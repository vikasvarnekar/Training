Views Menu (Item Window)

The Views menu provides options to view the access rights associated with the Item, history of the Item, revision history, and so on. The following table explains each option available in the Views menu.

Some options in the Views menu may appear disabled.  The menu options are enabled on appropriate Item selection and if the logged-in user has adequate access rights.  The access rights are configured by the administrator.

Menu Option

Description

Permissions --> View

Displays the Permission Item that currently applies to the open Item. The Permissions Item displays the name of the Identity and the rights and privileges assigned to the Identity for the open Item.

Permissions  --> Create Private

Creates a new Permission Item based on the existing Permission Item and user changes/inputs.

Permissions --> List of Allowed Permissions

Displays the list of allowed permissions for the Item. Changes the applied permission for the Item to the selected permission.

History

Displays the Item history. The Item history is displayed only if the administrator has enabled history tracking for the ItemType.

Revisions

Displays the different versions (generations) and revisions of the open Item.

Where Used

Displays a list of Items that reference the open Item.  The Items are displayed in a expandable tree-structure format. Clicking twice on the Item in the tree-structure opens the Item window.  For more information, refer to Where Used.

Structure Browser

Displays relationship instances in an expandable format.  For more information, refer to Structure Browser.

Workflow

Displays the Workflow Process (if one exists) associated with the open Item.

LifeCycle

Displays the current state of the open Item in the LifeCycle Map.
