About Files

The Files ItemType is a core ItemType used for managing the storage of files in Innovator vaults.  It retains the name of each file stored, the vault in which it is stored, the file type, and the user who has checked it out.

It is important to note that the Files ItemType should not be accessed directly as a means of storing files into the vault.  To store items in the vault, users should attach a property of Datatype Item, Data Source File to the ItemType to which the file is to be associated.
