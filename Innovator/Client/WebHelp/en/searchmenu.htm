Search Options

The Search menu provides various options to perform search related tasks.  The following table explains each option available in the Search menu.

Search Option

Description

Executes the search.

Advanced Search

Allows search on the properties that are not shown in the grid.  For more information, refer to Advanced Search.

AML Search

Allows use of complex search, which is created by the administrator.  For more information, refer to AML Search.

Hidden Search

Hides the blue search bar and prohibits you from specifying search criteria.

Simple Search

Allows you to perform search by specifying criteria for various properties in the blue search bar.  For more information, refer to Simple Search.

Save Search...

Allows you to save the current search criteria and retrieve previously created search criteria.  For more information, refer to Saved Searches.

Delete Saved Search

Deletes the selected saved search from the main grid.

Use wildcards

Allows usage of wildcards * an % in the search criteria.  Usage of wildcards is enabled by default.  If disabled, usage of wildcards is prohibited.

Append results

Allows you to specify another search criteria and append results to the items already displayed in the main grid.
For example, you can use append results to view Items with multiple but specific part numbers.

Sort across pages

Return multiple pages sorted as per sequence defined in ItemType.
