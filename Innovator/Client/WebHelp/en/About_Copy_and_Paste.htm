About Copy and Paste

The Copy and Paste function allows for efficient production usage of Innovator 12.0.  Items can be copied from within the data model and pasted into compatible items. Additionally the function allows the user to:

  • Copy one or more (multi-selected) items from within the relationship grid.
  • Paste the contents of the clipboard into the same or different source item of the same
  • View contents of clipboard in a pop-up window that identifies the source item, relationship type, and related item in the grid.
  • Maintain a clipboard that contains items copied from a different source types wherein the paste process has determined whether or not the pasted items are compatible with the target source item.
