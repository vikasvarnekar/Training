Preferences

Individual Users my choose how each ItemType is displayed in the standard user interface.  The preferences are stored in the database on log out and are loaded when the User logs in again.

A User can reset his/her preferences to the default values from the Main Menu by selecting Tools --> Preferences --> Reset User Preferences

Preferences can be viewed and edited by members of the Administrators Identity in the TOC Administration category.  The World Identity also has global preferences for Program Management.

When viewing Items in the main grid or related items in the relationships grid of an Item window the following changes can be made:

Columns may be hidden or inserted by right clicking the column heading row and selecting 'Hide Column' or 'Insert Columns ...' from the context menu

Columns may be resized by dragging the 'splitter' between column headings.

The order of columns can be changed by dragging a column heading to the left or right.

Default values for search result setting can be set
