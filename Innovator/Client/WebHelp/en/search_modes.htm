Search Modes

There are four search modes that you can select from the Search menu and the Search toolbar for both the main grid and the relationship grid. The modes are:

  • Hide Search Criteria: When you select this mode, the search bar does not appear. This mode prohibits you from changing the search criteria. Select another search mode to specify new search criteria.

Note: Hide Search Criteria does not remove or delete the previous search criteria, it only hides the display of the search criteria.

  • Simple Search: This mode is the most commonly used search mode and the easiest to use and understand. The blue search bar enables you to enter search criteria using wildcards. For more information, refer to Simple Search .

  • Advanced Search: This mode is a wizard that enables you to perform searches using properties and search conditions that cannot be used in simple search. For more information, refer to Advnced Search.

  • AML Search: This is a raw search mode t hat allows you to make create complex searches using attributes and properties. For more information, refer to AML Search.
