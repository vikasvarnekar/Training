Context Menu (Related Item)

When you use the right-mouse button (right-click) on a related Item in the relationship grid, the context menu is displayed in a pop-up form.  The following table explains each option available in the pop-up menu.

Note: The options displayed on clicking right-mouse button (right-click) differ as the options depend on the ItemType selected in the relationship grid.  The options for File ItemType may have additional options related to check-in/check out of the file.

Menu Option

Description

Open

Opens the Item window for the Relationship Item only if there is a Form defined for the Relationship Item.

For example, in Part ItemType, clicking the Open menu option in the relationship grid displays the Part-BOM Relationship Item.

Edit

Claims (locks) the related (child) Item in edit mode only if a related (child) Item is added to the relationship. A green flag appears in the relationship grid.

For example, in Part ItemType, clicking Edit in the relationship grid displays the related (child) Part Item.

Replace

Replaces the selected Item in the Relationship grid with the item you specify. The name of the replacement item appears in the Name column.

Claim

Locks the related (child) Item and prevents other users from submitting changes.

Actions

Opens the Search window to search an Item and replaces the current related/child Item with the Item selected from the Search window.

Relationship Types

Opens another context menu that enables you to select one of the following options:

  • Open the selected item.
  • Copy the selected row.
  • Paste the selected row.
  • Copies the selected item to the clipboard.
  • Displays the selected item in the Clipboard.
  • Promotes the selected item to the next stage.

Note: The context menu comprises standard options (covered in table) as well as Actions.  The administrator defines the available Actions for a relationship.  The context menu options for the various Actions to be displayed depend on the logged in user's access rights.  The access rights are configured by the administrator.
