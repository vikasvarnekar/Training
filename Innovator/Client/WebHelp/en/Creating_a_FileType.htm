Creating a File Type

  1. Select Administration>File Handling>FileTypes in the Navigation pane. The following menu appears:

  1. Click Create New FileType. A new FileType window appears:

  1. Enter the filetype information in the appropriate fields. The Ext. and Mime Type fields are required.
  2. Click the Select Items button  on the View With tab to associate an existing Viewer with the file. The Viewer Search dialog box appears.
  3. Click the Run Search  button to see a list of available viewers.
  4. Select a Viewer from the list and click the Return Selected  button.
  5. Click  and  to save and unclaim  the file.
