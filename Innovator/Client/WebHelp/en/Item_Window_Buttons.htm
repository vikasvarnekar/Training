Item Window Buttons

The buttons that appear in an Item window provide options to view the access rights associated with the Item, history of the Item, revision history, and so on. The following table describes the buttons and their associated options.

Some options may appear disabled.  The menu options are enabled on appropriate Item selection and if the logged-in user has adequate access rights.  The access rights are configured by the administrator.

Click...

Menu Option

Description
Permissions>View Displays the Permission Item that currently applies to the open item. The Permissions Item displays the Identity name and the rights and privileges assigned to the Identity for the open item.
Permissions>Create Private Creates a new Permission Item based on the existing Permission Item and user changes/inputs.
Permissions>List of Allowed Permissions Displays the list of allowed permissions for the Item. Changes the applied permission for the Item to the selected permission.
Copy ID Copies the Item ID to your clipboard.
Copy Link Copies the Item link to your clipboard.
Copy Link To... Copies the specified link to either the latest Item Version or the latest Item Release to the clipboard.
Export to Excel Exports the Item information to an Excel spreadsheet.
Export to PDF Exports the Item information to a PDF file.
Export to Word Exports the Item information to a Word document.
Multilevel BOM Report Creates a Bill of Materials report for an Item.
BOM Quantity Rollup Report Creates a report containing information about the items contained in the BOM and their quantities.
BOM Costing Report Creates a report that contains Cost and Cost Basis information for the BOM.
PE_SearchPartCircularReference Creates a report that lists the circualr dependencies between parts.
History  Displays the Item History.
Versions Displays the different versions (generations) and revisions of the open item.
Where Used Displays a list of items that reference the open item. The Items are displayed in an expandable tree structure format. Clicking twice on the Item in the tree-structure opens the Item window. For more information, refer to Where Used.
Structure Browser Displays the relationship instances in an expandable format. For more ifnormation, refer to Structure Browser.
Workflow Displays the Workflow Process (if one exists) associated with the open item.
LifeCycle Displays the current state of the open item in  the LifeCycle Map.
Promotes the item to the next stage in the process.
Refreshes the view.
Saves changes to the Item.
Saves changes to the Item and unclaims it.
Discards any changes you have made to the Item.
