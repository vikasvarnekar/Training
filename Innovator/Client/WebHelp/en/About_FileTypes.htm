About File Types

FileTypes define the viewer processing rules for files managed by the Innovator vault.  The FileType Items are used to pass the desired mime type association with a file extension to the web browser.  The browser uses the mime type association, and launches the application or plug-in which is associated with the mime type.

Additionally, an explicit application may be defined to be associated with a file extension using the View With tab.  The View With tab can direct the file to a specific application to be used for file viewing.
