Edit Menu

The Edit menu provides various options to perform editing related tasks for an Item such as copy, paste, edit, delete, unlock, and so on.  The following table explains each option available in the Edit menu.

Some options in the Edit menu may appear disabled.  The menu options are enabled on appropriate Item selection and if the logged-in user has adequate access rights.  The access rights are configured by the administrator.

Menu Option

Description

Purge

Deletes the selected version of the Item.

Delete

Selecting Purge this version deletes the selected version of the item. Delete all versions deletes all versions of the selected item.

Edit

Opens the selected Item in 'edit' mode.

View

Opens the selected Item in 'view-only' mode.

Copy

Allows you to copy the selected Relationship Item.  For more information, refer to Copying an Item.

Paste

Allows you to paste the copied Relationship Item.  For more information, refer to Copying and Pasting Relationship Item.

Paste Special

Allows you to paste the copied Relationship Item with the ability to copy permissions and/ or create child Item.  For more information, refer to Using Paste Special.

Show Clipboard

Displays the list of Relationship Items copied and available on the clipboard.

Claim

Locks the selected Item and prevents other users from submitting changes.  For more information, refer to Claiming or Unclaiming an Item.

Unclaim

Unlocks the selected Item and allows any authorized user to lock item for change.  For more information, refer to Claiming or Unclaiming an Item.

Undo Changes

Reverts Item back to values since it was last saved.

Promote

Promotes Item to next state defined in assigned Lifecycle map.  For more information, refer to Promoting an Item in Lifecycle.

Versions

Displays the version history of the selected Item.
