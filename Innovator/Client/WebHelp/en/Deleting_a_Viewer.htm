Deleting a Viewer

  1. In the Navigation pane, select Administration>File Handling>Viewers. The following menu appears:


  2. Click Search Viewers. The Search grid appears.
  3. Click  to see the list of available viewers.
  4. Right click the appropriate viewer row.  The following context menu appears.

  5. Click Delete All Versions. A pop-up window similar to the following appears:

  6. Click OK.






Note:  When an error message appears after the confirmation pop-up window, the item has not been deleted. The name of the property corresponding to the ItemType appears in the error message as [itemtype_name property_name] and must be deleted from ItemTypes.
