Form Fields

The following fields are available for use on forms:

Field Type

Description

Quick Access Button

Text

Standard text field

Password

Text fields where the text is hidden via MD5

Text Area

Standard text area with a scroll bar

Check box

Standard check box

Formatted Text

A text area with basic formatting options (alternative to text area)

 

Label

Standard label field

 

Button

Standard button

Radio Button List

Standard set of radio buttons (alternative to dropdown list)

Dropdown

Standard dropdown list

List box Single Select

Standard list box with a scrollbar (alternative to dropdown list)

List box Multi Select

Standard list box with a scrollbar capable of multiple selections

Check box List

Standard set of check boxes (alternative to list multi select)

 

Date

Text field with a popup calendar to select a date

Color

Field with a popup color selector

Color List

A dropdown list of colors

 

HTML

A field that can contain custom HTML code including scripts

Item

A text field with an ellipsis button next to it, enabling a search for another item

 

File Item

A field that enables the user to browse and select an attachment file

 

Class Structure

A filed that enables the user to classify the item

 

Image

A fields that enables the user to select an image

 

Multilingual String

A text field that enables the user to enter multiple entries in different languages

 

Group Box

A field to group other fields together
