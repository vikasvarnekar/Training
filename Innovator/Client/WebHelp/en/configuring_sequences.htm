Configuring Sequences

Sequences are used for automatic numbering of PRs, ECRs, and ECNs, or any other chosen item in Innovator.  Sequential numbering is an option for Parts and Documents.  This section shows how to configure the appearance of a sequence to comply with your company's requirements.

To see the pre-defined sequences:

  1. Select Administration -->Sequences-->Search Sequences in the Navigation pane.
  2.  Click . The Search grid displays a list of existing sequences:

Here is a list of the properties and their contribution to the sequence configuration:

Name - the name of the sequence item.

Prefix - the alpha-numeric prefix of the generated number.  For example, by default, all ECNs will have an ECN number: ECN-10000x.

Current Value - the current value of the sequence.

Suffix - similar to Prefix, except added to the end of the number.  For example, if you were to specify a suffix of -GTX, the ECN number would look like: ECN-100001-GTX.

Pad With - specify the character to fill in the required number of spaces

Pad To - the total number of characters for the number, excluding the prefix and the suffix

Initial Value - the increment by which the number differs from the previous and next.
