Claiming or Unclaiming an Item

You can lock or unlock an Item if you have the required access rights.  When you lock an Item, it prevents other users from submitting changes to the locked Item.  After you unlock the Item, you or another user may lock the Item to submit additional changes to the Item.

Locking an Item

There are multiple options to lock an Item.  From the main grid, select the Item to be locked, and then opt for one of the following choices:

  1. Click on the main toolbar.
  2. Click on the main toolbar.
  3. Navigate to Edit --> Lock.
  4. Use right mouse button (right-click) to open the context menu and click Lock from the context menu.
  5. Use right mouse button (right-click) to open the context menu and click Edit from the context menu.
  6. In an open Item window, you can click on the Item toolbar.
  7. In an open Item window, you can navigate to Edit --> Lock from the Item menu.

Lock Status

The following table lists the different locks and its description:

Icon

Description

Locked by you.  Other users are prohibited from submitting changes for the Item you locked.

Locked by you, but Item has undergone changes that are not saved or submitted.  You need to save the Item to submit the changes.

Locked by another user.  You cannot unlock or edit an Item locked by another user.

 

Unlocking an Item

There are multiple options to unlock an Item. From the main grid, select the Item to be unlocked, and then opt for one of the following choices:

You can only unlock Items that are locked by you. You cannot unlock an Item that is locked by other user.  Administrators may override the lock and unlock the Item.

  1. Click on the main toolbar.
  2. Navigate to Edit --> Unlock.
  3. Use right mouse button (right-click) to open the context menu and click Unlock from the context menu.
  4. In an open Item window, you can click on the Item toolbar.
  5. In an open Item window, you can navigate to Edit --> Unlock from the Item menu.
