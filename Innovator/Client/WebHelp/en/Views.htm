Views

Views are a relationship between an ItemType and a Form.

Alternative forms can be displayed for Adding, Viewing and Editing, which is selected in the Type column. Default means both Viewing and Editing different Identities, selected in the Identity column.Where an ItemType uses a class structure, alternative forms can be used for a Classification. If more than one Form is qualified for a particular situation, the Form with the highest priority will be displayed.
