Creating E-Signature

An E-signature is separate from a password. E-signatures are user-controlled passwords maintained by users, not an Innovator Administrator or IT password.  This allows complete control over an E-Signature by each user.

1.  Log on to Aras Innovator.

2.  Navigate to Tools --> Preferences --> Change E-Signature. The Change E-Signature dialog box appears.


3. Type your current E-Signature in the Old Password field.

4. Enter your new password in the New Password and Confirm New Password fields.


5. Click OK.

If you are setting the E-Signature for the first time, leave the Old Signature field blank.
