Relationship Toolbar

The relationship toolbar includes various buttons to perform various tasks on the relationship grid. The various buttons in the relationship toolbar are listed below:

Button Name

Button

Description

Create Item

Creates a new Relationship Item.

All the three options may not be always enabled as it depends on the ItemType configuration done by the administrator. Â Depending on the configuration done by the administrator, either or all of the Actions are enabled and available.

Select Items

Opens the Search dialog window to search for the related Item and replace the selected Item with the searched Item.

Delete Row

Deletes the selected relationship(s).

Run Search

Runs specified search.

Add Criteria

This button appears when you select the Advanced search mode. Click it to add search criteria.

Refine

Displays a list of properties and xProperties associated with the item. Select the appropriate properties and xproperties to use as criteria.

Export to Excel

Select Export to Excel from the context menu. Exports all the properties of the related (child) Item listed in the relationship grid to Microsoft Excel sheet.

Export to Word

Select Export to Word from the context menu. Exports all the properties of the related (child) Item listed in the relationship grid to a Microsoft Word document.

Claim

Locks the selected child (related) Item and prevents other users from submitting changes.

Unclaim

Unlocks the selected child (related) Item and allows any authorized user to lock Item for change.

Promote

Provides options to promote the relationship Item to the next available state, defined in the assigned Lifecycle map.

Promotes the Relationship Item and NOT the child or related Item.

Run Search

Executes the search.

Clear Search Criteria

Removes the applied search criteria.

Search Mode drop-down

Allows you to select the search mode. Â 

Display

Enables tab redline view.
