Components of a Message

Once the message is shown within the area of the discussion panel, there are various components in it that have specific properties.

The components of a message are shown below:

 

 

The table provides the components of a message:

 

Components of Message

Properties

Source Icon

Indicates whether the source of the message is from a user comment or automatic.

User name

Name of the user who created the comment, either directly or indirectly through History.

Creation time

Time that the message was created

Board Item

The Board item shows the name of the item who’s Board the message is related to. The Board Item also acts as a hyperlink to open the tear-off window of the referenced item.

Item info

The Item Info shows the basic item information from the item at the time the message was created. The displayed string is a combination of three properties from Secure Message as follows:

  • Current revision number of the item

  • Current generation of the item

  • Current lifecycle state of the item

Item Icon

Icon and color corresponding to the board item

Message text

The text of the message.

Thumbnail

The thumbnail of the image (if included in the comment)

Message functions

 

The functions that can be performed on the message.

Functions that can be performed on the message, including:

  • Flag/Unflag -  Users can flag/unflag a message. It also displays the number of users who have flagged/unflagged the message.

  • Reply - Allows the user to enter a comment in reply to this message

  • Hide replies - Allows the user to hide the display of reply messages.

  • Erase - Allows the user to erase a previously-created message

 

Display of Messages

The number of displayed Messages is controlled by the Default Page Size property of the Secure Message ItemType.

Use the scroll keys to scroll up and down to the bottom of the displayed messages. Click on "Show more messages" located at the footer of the discussion panel to display more messages.

 

  

When all the messages have been displayed, the footer of the discussion panel indicates "There are no more messages".
