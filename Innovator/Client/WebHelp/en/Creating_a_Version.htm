Creating a Version

When you create or edit an item, you create a version of that item. In this example, we will use Part J-100002.

  1. In the Navigation pane, select Design>Parts.  The following menu appears:

  1. Click Search Parts.  The Parts search grid appears.

  1. Right click on the J-100002 row and click Versions in the context menu that appears. The Item Versions pop-up box appears.

  1. Click  to open the item for editing.
  2. Click the Select Items button  on the BOM tab. The Part search dialog appears.

  1. Select a part to add to the BOM and click the Return Selected button .

  1. Click  to save and close the item.
  2. Click the Navigate button  and select Versions from the context menu. The Item versions pop-up displays the new version information.
