Copying and Pasting a Relationship Item

You can use the Copy and Paste functions to create multiple instances of a Relationship Item. In this example, we will use the Part ItemType as the source item.

Creating a copy of a Relationship Item

  1. Log on to Aras Innovator.
  2. Select Design>Parts>Search Parts. The Search grid appears.
  3. Right click the source (parent) Item for which you want to add a relationship. The part item window appears.

4. In the relationship grid, right-click the Item to be copied.  For example, let us consider there are multiple nuts related to a chair.

5. Select Save As from the context menu. A copy of N-1002 appears.

 

Note: You can only copy Items that are saved to the Server.  If you try to copy items that have not been saved, an error appears.
