TOC View

Information in this tab defines what the user sees when they select an Item from the grid.  For example, when we select Design>Parts from the Navigation pane and either click the Search icon  or the Search Parts button, the Item Grid appears:

 

 

However, if you select Administration--> Configuration -->Dashboard--> Design To Goal , you will see dashboard metrics, not the item grid.

 

This is so, because in the ItemType definition of the Design To Goal dashboard, the TOC View is defined as a web page, as follows:

 

To create your own TOC View, fill in the following properties:

 

  • Name - the identity for whom this TOC View applies; different identities can have different views
  • Start Page - the actual http path to the designated web page
  • Parameters - any inputs that are necessary or expected by the web page
