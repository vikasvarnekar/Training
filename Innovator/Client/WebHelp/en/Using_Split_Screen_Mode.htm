Using Split Screen Mode

Split Screen Mode enables users to enable or disable a side-by-side view by clicking the Split Screen button in the header:

 

 

The following example shows a split screen view with the Part search grid on the left and the individual Part items on the right.

 

 

The Part Search grid is docked. The tabs on the right are undocked, and the active tab is displayed. Clicking a tab in the left pane opens the item in the right pane.

 

Using the Context Menu

Right-click either pane to view the context menu.

 

 

The context menu lets you close a specific tab, selected tabs, tabs that appear on either the right or left side of the screen, or all tabs. Click Tear Off Tab close all tabs. Clicking Dock Tab enables you change a tab from undocked to docked.

 

Changing Split-Screen Proportions

You can change the proportions of the split pane view by clicking and dragging the pane divider.

 

 

Hover over the pane divider. A double-headed arrow appears and the pane divider changes color to indicate that the pane proportions can be changed.

 

 

Slide the divider to change the proportions.
