Item Window

An Item window, also known as a Tear Off window, contains information about an Item. When you edit an Item, an Item View appears in a Tear Off Window based on the Form that has been created for that ItemType. Tear Off windows are used for data entry and viewing detailed information beyond what is on the main search grid.

An Item window consists primarily of the following:

Forms

Item Menu

Item Toolbar

Relationships

Within a form, you can search for values contained in either a dropdown menu or a dialog box. Sinply enter a letter in one of these fields to see a list of options that start with the same letter. For more information, refer to Using Typeahead.

The Form and the Relationship tabs vary by ItemType since they are highly customizable.

If an Item has its corresponding ItemType enabled for social content, then the User Interface for the Item Window will have a few added special properties.

Refer to Secure Social Visual Collaboration for Item Windows (SSVC) for more information on this.

If you have several ItemType windows open at one time, you can rearrange the tabs using drag and drop.
