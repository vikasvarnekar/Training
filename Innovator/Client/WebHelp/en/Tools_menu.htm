Tools Menu

The Tools menu contains administrative operations as well as user preferences settings.  Users may choose how each ItemType is displayed in the user interface.

The following table explains each option available in the Tools menu.

Menu Option

Description

Preferences --> Change Working Directory

Allows you to change the working directory, which is the local location for files that are retrieved from the system (Check out).  For detailed steps on changing working directory, refer to Setting Up Working Directory.

Preferences --> Show Toolbar Text

Displays text alongside the main toolbar buttons.

Change Password

Allows you to change your login password. For more information on how to change your password, refer to Changing Password

Change E-Signature

Allows you to change your E-signature.
An E-signature is separate from a password.  E-signature is a user-controlled password maintained by the user, not administrator, or IT password.  This allows complete control over an E-Signature by each User.  For more information on how to create a E-Signature, refer to Creating E-Signature.

Refresh

Reloads the page and refreshes the current view.
