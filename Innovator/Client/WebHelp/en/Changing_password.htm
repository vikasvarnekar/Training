Changing Your Password

To access Aras Innovator, a user must be defined in the system with credentials (user name and password).  You can change your password using the the following procedure:

  1. Log into Aras Innovator
  2.  Click the User Menu    button in the upper right corner of the window. The User Menu appears.
  3. Select Preferences>Change Password. The Change Password dialog box appears.


  4. Enter your current password in the Old Password field.
  5. Enter the new password in the New Password field. Re-type your new password for confirmation in the Confirm New Password field.
  6. Click OK.
