Vault Storage Locations

The vault location for stored files is determined by a User setting that is established independently for each individual user.  The storage location for the user is used regardless of where previous versions of the file may reside.  For example, if a user who is set up to store files on Vault B checks out a file from Vault A, when the file is checked back in it will go to Vault B.  This means that if versioning is turned on, different versions of the file may end up residing in different vaults.
