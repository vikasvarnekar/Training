Setting Client Display Mode

Use the following procedure to change the way that information is displayed in Aras Innovator:

1. Select Administration>Variables>Search Variables.

2. Enter C* in the Name field and click . A list of variables appears.

 

3. Double-click on Client Display Mode. The associated Variable window appears.

 

4. Click  to change the Value and Default Value fields from Tabs (Tab Mode) to Windows (Standard Mode).

    Tab Mode looks like this:

 

 

If you specify  Standard Mode, the item you select appears in a separate window.

5. Click to save your changes.
