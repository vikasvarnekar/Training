Dynamic Assignments

Dynamic assignments must be set up in a Workflow Map

Select the Manager of any Activity for which you wish to implement dynamic assignments. Only the Manager will be able to change assignees for a Workflow Process

Select the Role for the same activity, the Role is an Identity; only members of this Identity may be assigned to this Activity

Manager and Role must be set for all Activities in the Workflow Map that will have dynamic assignments.

 

Dynamic Assignments are executed in a Workflow Process

To make a dynamic assignment

Open the source Item to which the Workflow Process is attached, in this example a Problem Report

In the Item window Menu select Views --> Workflow

 

d the following window will open, and lock the Workflow Process

 

Select the Activity to for which you wish to change assignments

If you are a member of the Managed By Identity, the Add and Delete Icons will be enabled

Add, Delete or Edit assignments

When you save the changes they will be validated to ensure that the assignees you selected are members of the Role
