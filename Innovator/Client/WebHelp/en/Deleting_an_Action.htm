Deleting an Action

  1. In the Navigation pane, select Administration>Actions. The following menu appears:

  1. Click Search Actions. The Search grid appears.
  2. Click  to view the list of available actions.

  1. Right click the appropriate action to access the context menu:

  1. Select Delete All Versions.  The following pop-up window appears:

  1. Click OK.

Note:  When an error message appears after the confirmation pop-up window the item has not been deleted.  The name of the property corresponding to the ItemType appears in the error message as [itemtype_name property_name} and must be deleted from ItemTypes.
