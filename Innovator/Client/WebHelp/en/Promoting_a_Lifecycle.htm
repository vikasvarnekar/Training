Promoting an Item in LifeCycle

LifeCycle maps are defined by an administrator and associated with an ItemType.  These maps provide rules for the progression of an Item through different LifeCycle states.  An Item can belong to only one state at a time and any change to a new state is called a Promotion.

If you try to promote an Item for which you do no own required access rights, an error message 'No Promotions Available' is displayed.

To promote an Item to a new LifeCycle state

  1. Log on to Aras Innovator.
  2. From the TOC, navigate to the Item. Let us consider Part ItemType.  
  3. Navigate to Design --> Parts.
  4. From the main grid, select the Item to be promoted.
  5. To promote the Item, use one of the following choices:
  1. Click on the main toolbar.
  2. Use the right-mouse button (right-click) to open the context menu and select Promote.
  3. Alternatively, you can open the Part Item window, and click on the Item toolbar or select Edit --> Promote from the Item menu.

Note:  You cannot promote a claimed Item. For an Item to be promoted, it should be unclaimed.  The Promote button on the toolbar or the option in the menu is enabled only for unclaimed Items.

The Promote dialog box appears with a list of states to which the Item can be promoted.

Note:  The To States displayed in the Promote dialog box are different for every Lifecycle Map.  The LifeCycle Map is configured by the administrator.

6. Select the required To State and click  .
