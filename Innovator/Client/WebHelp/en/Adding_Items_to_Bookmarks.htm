Adding Items to Bookmarks

Any Secure Social enabled items can be added two places in the Bookmarks. Either to the Items in the Bookmark Groups or to a Forum.

To add item(s) to Bookmarks, do the following:

  1. Right click on any of the SSVC enabled items that you want to add.

  2. Click on Add Items to Bookmarks in the context menu.

This opens the Add item(s) to Bookmarks dialog box.

  1. Select one of the following options:

    • Item Bookmarks - adds the item you select to your Bookmarks list.

    • Create new Forum- Select this option to access the Create New Forum dialog box which is described in the Add Forum section. When you create a new Forum, the selected item will be placed in it.

    • Bookmarks - Selecting this option will add the item to the Items group in Bookmarks ..

    • The third option is an existing forum owned by the user. Selecting this option would add the selected item to that Forum. It is not possible for a user to add an item to a Forum they do not own.

  2. Click OK to save the changes or click Cancel to exit from the dialog box.
