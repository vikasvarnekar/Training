Context Menu  (Right-Click Menu)

When you use the right-mouse button (right-click) on an Item in the main grid, the context menu appears. The items contained in the context may vary depending on the grid type. The following example shows the ItemType search grid:

 

.  The following table explains each option available in the context menu.

The options displayed on clicking right-mouse button (right-click) differ as the options depend on the Item selected in the main grid.

 

 

Menu Option

Description

Open

Opens an Item.

Claim

Locks an item for editing.

Promote

Provides options to promote the Item to the next available states, defined in the assigned Lifecycle map.  For more information, refer to Promoting an Item in Lifecycle..

Properties

Displays generic properties of the Items such as date of creation, modification, creator, modifier, and so on.  It also provides an option to copy the 32 digit unique ID assigned to the Item.

Save As

Creates a copy of the item.

Delete All Versions

 Deletes all the versions associated with the selected Item.

Structure Browser

Displays the structure of the selected item.

Where Used

Displays a tree structure that shows the relationships between items.

Revisions

Displays the revisions of an item.

Add to Package Definition

Enables you to select a package definition to add the item to. You can also create a new package definition for a particular item.

Reset Lifecycle State

Logged when the "reset_lifecycle" server action is called. This is normally done by an administrator. No comment is stored.

Reset Item Access

Enables you to reset access rights for an item..

Set Default Life Cycle

Enables you to create a LifeCycle map for an item.

ItemType Definition Report

Displays the information about the specified Item.

ItemType Permissions Report

Displays permissions information for the selected Item.

ItemType History Configuration

Displays information about properties and actions associated with the Item.
