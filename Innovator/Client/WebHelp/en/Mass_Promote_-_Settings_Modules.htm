Mass Promote - Settings Modules

Using the Life Cycle Map Module

The Life Cycle Map module enables you to:

  • Select a life cycle map to use for the items you want to promote.

  • Select the target state for all the items that you are promoting.

  • Enter a comment that will be applied to all of the items that you are promoting.

  • Remove invalid items from the grid using the Using Invalid Items module.

  • Promote item groups using the Execute module.


This module only appears in the Mass Promote window if the items you selected do not share the same Life Cycle Map. You can only promote items that share the same Life Cycle maps. Items that use a map that you did not select are flagged as invalid by Mass Promote.

Selecting a Life Cycle Map

Use the following procedure:

  1. Click the down arrow in the Select Life Cycle dropdown to see a list of life cycle maps. If there is only one map associated with the items, the map name appears.

  1. Select the appropriate Life Cycle map.

 Using the Target State Module

Selecting a Target State

The Target State module always appears in the window but is disabled until you select a Life Cycle map to use with the items you selected previously. Once the Target State module is active, use the following procedure to select the target state that you want to use for the items you are promoting:

  1. Click the down arrow in the Target State dropdown to see a list of the Target states associated with the Life Cycle map you selected.

  1. Select the appropriate Target State and click Apply. Some items may become invalid, either because they do not have a path to the Target State you selected, or you do not have the appropriate permissions to promote them.

Using the Invalid Items Module

Resolving Invalid Items

The Invalid Item module appears when the application detects one or more invalid items in the list that you selected. Use one of the following options to remove invalid items from the list:

  • Select the Remove  radio button and click Apply to remove all invalid items from the grid. The Status Module is updated.

  • Select the Move to another promote session radio button and click Apply. All of the invalid items are removed from the grid and the Status Module is updated. A new Mass Promote window opens displaying a list of the invalid items.

Note:  You can perform these operations manually in the grid by selecting individual items and then picking either the Remove From Selection or the Move to a New Mass Promotion radio button.

Using the Execute Module

Promoting Items

The Execute Module enables you to promote the items you select. It appears in the Mass Promote window at all times. Different buttons are activated, depending on the current state as shown in the following table.

Button

Status

Action

Conflicts not resolved

Closes the MassPromote window without executing any promotions.

Execution in progress

Pauses the current execution process.

Execution in progress

Starts the promotion process for all selected items.

Execution Complete

Saves the Mass Promotion items and closes the window.

Apply changes.

You can choose to either remove Invalid items or move them to another promotion level, depending on the radio button you select in the All Invalid Items module. The Apply button is activated when you make your selection.


Note:  When you click the Cancel button to close the Mass Promote window without executing the Promote action, a dialog box containing a warning prompt appears. Clicking Yes closes the Mass Promote window. No promotions are executed and the selected items are not saved. Clicking No leaves the Mass Promote window in the current state.

When you click the Cancel button during promotion execution, a dialog box containing a warning prompt appears. Clicking Yes halts the Mass Promote process. You can view the partial results and click Done when you are finished. Clicking No causes the promotion process to continue.
