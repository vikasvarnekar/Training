About Viewers

Viewers define applications that are launched in a web browser to view files stored in the Innovator vault.  The Viewer URL links to an HTML file that contains the launch instructions for the viewer.  A Viewer may be associated to FileType items under the View With relationship tab.  

When a file is opened for viewing, the Viewer is launched and passed the file URL as input.  When no Viewer is associated with a FileType, the Web Browser will attempt to open the file using mime type or plugin-controlled associations.
