About Mass Promote

The Mass Promote feature in Aras Innovator enables users to promote multiple items at one time by selecting them from the Search Results grid and clicking the Promote icon on either the toolbar or the context dropdown menu.

The Mass Promote window appears as either a new tab (tab mode) or tear-off window (standard mode).

Notes:
Promote is enabled regardless of the types of items you select. Verification checks are not made until the Mass Promote window appears.

You can only select multiple items that have the same itemtype when using Mass Promote.

The Mass Promote Window         

The Mass Promote window consists of the banner, the settings panel, and the grid. The banner displays the number of selected items and their itemtype. The settings panel displays the following modules:

  • Life Cycle Map only appears when the items that appear in the grid do not share the same life cycle map. You need to select a life cycle for the items that you want to promote. Items that do not have a life cycle map associated with them are considered invalid by the system.

  • Target State enables you to select a target state for the items that you are promoting. Items get promoted to the state you select.

  • Comment only appears when comments are enabled for a particular Life Cycle transition for one or more items associated with the specified Target State.

  • Invalid Items only appears if one or more invalid items are detected in the selected items list.

  • Execute appears once conflicts between items are resolved.

  • Status displays the current promotion status.
