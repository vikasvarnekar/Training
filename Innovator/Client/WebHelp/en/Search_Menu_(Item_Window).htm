Search Menu (Item Window)

The Search menu provides various options to perform search related tasks in the relationship grid.  The following table explains each option available in the Search menu.

Menu Option

Description

Run Search

Executes the search in the relationship grid.

Advanced Search

Allows search on the properties that are not shown in the relationship grid.  For more information, refer to Advanced Search.

AML Search

Allows use of complex search, which is created by the administrator.  For more information, refer to AML Search.

Hidden Search

Hides the blue search bar in the relationship grid and prohibits you from specifying search criteria.

Simple Search

Allows you to perform search by specifying criteria for various properties in the blue search bar of the relationship grid.  For more information, refer to Simple Search.

Save Search...

Allows you to save the current search criteria and retrieve previously created search criteria.  For more information, refer to Saved Searches.

Delete Saved Search

Deletes the selected saved search from the relationship grid.
