Erasing a Message

Any existing message can be erased only by the creator of the message or by an administrator. To erase a message, do the following:

1, Click on the Erase link found below the message that needs to be erased.

A dialog box similar to the following appears.

  1. Click OK to delete the message or Cancel to exit from this dialog box.

  2. Once the message is erased, new text is shown in place of the removed text. The context of the message is: Message erased .

It is not possible for a user to erase a message they have not created. Messages created by other users do not contain the Erase link.

Note: Messages of type “History” cannot be erased.
