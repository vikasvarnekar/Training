CUI

Configurable User Interface（CUI）-UIコントロールの構造を表す特定のデータモデルと、特定の場所に固有のビジネスロジック（？）、および現在のプラットフォームに固有のコントロールのセットを組み込む技術。

CUIというのは、「ハードコーディングされた」レイアウトではなく、いわゆるレイアウトに結合された制御定義の構造化モデルによって定義されたユーザーインターフェイスのことです。 このモデルは、コーディングなしでビジネス目標に応じて簡単に変更できます。

このような概念は、3つの主要なコンポーネントを意味する:

  1. データモデルとそのルール
  2. 提供されたデータ(CUI LayoutsおよびCUI Controls) に従ってUIを構築するCUIエンジン
  3. どこでも使用でき、CUIエンジンによってデータとビジネスロジックを取り込むことができる純粋なコントロール(components)

関連するファイルは次の場所にある: Innovator.git\Innovator\Client\Modules\cui\.
