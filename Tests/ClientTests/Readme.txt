This file is "read me" file for Client Unit Tests in Customer Repository Template.

Tests are located in the 'tests\' folder. To run them use 'RunClientUnitTests.bat' 
in the root CRT folder. To know more look a presentation in the 'Documentation\For Developers' 
fodler.

Notes:
	- You can check code coverage in folder 'Tests/ClientTests/reporters/coverage'.
	- To learn more about karma.conf.js refer to http://karma-runner.github.io/2.0/config/configuration-file.html
